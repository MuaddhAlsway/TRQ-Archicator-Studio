QUICK START (5 MINUTES)
======================

1. Login to Cloudflare:
   npx wrangler login

2. Set Turso secret:
   wrangler secret put TURSO_AUTH_TOKEN --config wrangler-workers.toml --env production
   (Paste token from server/.env)

3. Deploy backend:
   npm run deploy:worker:prod

4. Deploy frontend:
   npm run deploy:prod

5. Verify:
   - Frontend: https://trq-studio.pages.dev
   - Backend: https://api.trqstudio.com/api/health

LOCAL DEVELOPMENT

Terminal 2 - Backend:

DEPLOYMENT COMMANDS
===================

Backend:
  npm run deploy:worker:prod      # Deploy to production
  npm run deploy:worker           # Deploy to staging
  npm run worker:dev              # Local development

Frontend:
  npm run deploy:prod             # Deploy to production
  npm run deploy                  # Deploy to staging
  npm run dev                     # Local development

Monitoring:
  wrangler tail --config wrangler-workers.toml  # View logs
  wrangler deployments list --config wrangler-workers.toml
