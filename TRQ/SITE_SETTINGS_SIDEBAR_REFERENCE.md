## ðŸ“‹ FIELD TYPES REFERENCE

### Text Input
Single line text field
- Examples: Titles, headings, button text
- Limit: Usually 100-200 characters

### Text Area
Multi-line text field
- Examples: Descriptions, paragraphs, biographies
- Limit: Usually 500-2000 characters

### Image URL
Link to an image
- Format: Full URL (https://...)
- Recommended: 1920x1080 or larger

### Select/Dropdown
Choose from predefined options
- Examples: Page links, icons, colors
- Can only select one option

### Toggle/Checkbox
True/False option
- Examples: Show/Hide sections
- Values: true or false

### JSON Array
Structured data format
- Examples: Categories, project types, form fields
- Must be valid JSON format

### Pipe-Separated List
Items separated by pipe character (|)
- Examples: Workflow features, form subjects
- Format: Item 1|Item 2|Item 3

## ðŸ”„ COMMON WORKFLOWS

### Update Home Page Hero
1. Go to Site Settings
2. Click "Introduction" tab
3. Edit Title, Text 1, Text 2, Image
4. Update Link Text and Link Page
5. Click "Save Changes"

### Add Featured Projects
1. Go to Site Settings
2. Click "Featured Projects" tab
3. Click "Add Project" button
4. Select projects from dropdown
5. Reorder by dragging if needed
6. Click "Save Changes"

### Update Contact Information
1. Go to Site Settings
2. Click "Contact Page" tab
3. Scroll to "Contact Information" section
4. Edit each block (Icon, Title, Details)
5. Toggle visibility with "Show" checkbox
6. Click "Save Changes"

### Customize Pricing Form
1. Go to Site Settings
2. Click "Pricing Page" tab
3. Edit Project Types, Sizes, Timeline, Budget JSON arrays
4. Edit Contact Fields JSON array
5. Update form labels and descriptions
6. Click "Save Changes"

### Update Blog Categories
1. Go to Site Settings
2. Click "Blog Page" tab
3. Edit category labels (Design Tips, Trends, etc.)
4. Update Featured Label and Read Article Text
5. Edit Newsletter section
6. Click "Save Changes"
