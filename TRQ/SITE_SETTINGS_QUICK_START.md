# Site Settings - Quick Start Guide
## Get Started in 5 Minutes

---

## üöÄ FIRST TIME? START HERE

### Step 1: Access Site Settings
1. Log in to admin panel
2. Look at the left sidebar
3. Click **"Site Settings"** (gear icon)
4. You'll see tabs at the top

### Step 2: Choose Your Page
The tabs show all 11 pages you can edit:
- **Introduction** - Home page hero
- **Featured Projects** - Home page projects
- **How We Work** - Home page workflow
- **CTA Section** - Home page call-to-action
- **About Page** - About page content
- **Services Page** - Services page content
- **Workflow Page** - Workflow page content
- **Portfolio Page** - Portfolio page content
- **Contact Page** - Contact page content
- **Pricing Page** - Pricing page content
- **Blog Page** - Blog page content

### Step 3: Edit Content
1. Click on a tab
2. Find the field you want to edit
3. Click in the text box
4. Type your changes
5. Repeat for other fields

### Step 4: Save Changes
1. Scroll to the top
2. Click **"Save Changes"** button
3. Wait for "Saved!" confirmation
4. Done! Changes appear on website

### Step 5: Verify on Website
1. Open your website in another tab
2. Refresh the page
3. Check that your changes appear
4. If not, clear browser cache and refresh again

## üìù EDITING BASICS

### Text Fields
- **Single line** - For titles, headings, button text
- **Multi-line** - For paragraphs, descriptions
- Just click and type

### Images
- Paste full URL (starts with https://)
- Example: `https://images.unsplash.com/photo-xxx`
- Recommended size: 1920x1080 pixels

### Dropdowns
- Click to open list
- Select one option
- Used for page links, icons, colors

### Buttons
- **Save Changes** - Saves all edits at once
- **Add Project** - Adds featured projects
- **Remove (X)** - Removes items from lists

## üîç FIELD GUIDE

### What Each Field Does

**Title**
- Main heading on the page
- Keep it short (under 60 characters)
- Make it attention-grabbing

**Description/Paragraph**
- Supporting text
- Can be longer (up to 500 characters)
- Explain the title

**Image**
- Visual element
- Must be full URL
- Should be high quality

**Button Text**
- What the button says
- Keep it short (2-4 words)
- Use action words (Learn, Explore, Contact)

**Button Page**
- Where button links to
- Choose from dropdown
- Must be a valid page

**Icon**
- Visual symbol
- Choose from dropdown
- Represents the section

## üé® FORMATTING TIPS

### For Lists (Pipe-Separated)
Use the pipe character (|) to separate items:
```
Item 1|Item 2|Item 3|Item 4
```

### For JSON Arrays
Use proper JSON format:
```json
[
  { "id": "all", "label": "All Projects" },
  { "id": "residential", "label": "Residential" }
]
```

### For Links
- **Internal pages:** Choose from dropdown
- **External links:** Use full URL (https://...)
- **Email:** Use `mailto:email@example.com`
- **WhatsApp:** Use `https://wa.me/966XXXXXXXXX`

## üÜò TROUBLESHOOTING

### Changes Don't Appear
1. Click "Save Changes" button
2. Wait for "Saved!" message
3. Refresh website page
4. Clear browser cache (Ctrl+Shift+Delete)
5. Try different browser

### Image Not Showing
1. Check URL is complete (starts with https://)
2. Try a different image URL
3. Verify image file exists
4. Check image size (should be large enough)

### Can't Find a Field
1. Make sure you're on the right tab
2. Scroll down to see more fields
3. Use browser search (Ctrl+F) to find text
4. Check if section is hidden/collapsed

### Accidentally Deleted Content
1. Edit the field again
2. Type the content back
3. Click "Save Changes"
4. (Note: There's no undo, so be careful!)
